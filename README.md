# Mangalam HDPE - Premium Industrial Solution

A high-fidelity, pixel-perfect web interface developed as part of the Gushwork technical assignment. This project replicates the professional and cinematic aesthetic of the Mangalam HDPE Pipes brand, focusing on interactive product displays and modern technical textiles manufacturing solutions.

## Key Features

- **Interactive Product Carousel**: A high-end gallery with thumbnail synchronization and individual image navigation.
- **Precision Magnification Lens**: Custom-built vanilla JS zoom functionality with coordinates-to-percentage mapping (2.5x magnification).
- **Intelligent Sticky Header**: A responsive navigation bar that adapts to the scroll position (appears beyond the hero fold, tracks scroll direction for visibility).
- **Responsive Industrial Grid**: Fully fluid and responsive layout optimized for Desktop, Tablet, and Mobile viewports.
- **FAQ Accordion System**: Seamlessly animated question-and-answer section with auto-closing logic for optimal UX.

## Tech Stack

- **HTML5**: Leveraged semantic markup for improved accessibility and SEO.
- **CSS3**: Utilized a modern architecture with CSS Variables (Custom Properties), Flexbox, and Grid layouts. Glassmorphism and grain filters were implemented for added texture.
- **JavaScript (ES6)**: Pure vanilla implementation with zero external dependencies or frameworks, ensuring maximum performance and lightweight execution.
